# LedgerLearn React App

## Setup
1. `npm install`
2. `npm run dev`

## Stack
- React + Vite
- Tailwind CSS
- React Router

## Routing
- `/` -> Home page (`src/pages/HomePage.jsx`)

## Animated Background Option
1. Add images in `public/backgrounds/`.
2. Set `ENABLE_ANIMATED_IMAGE_BG` to `true` in `src/pages/HomePage.jsx`.
